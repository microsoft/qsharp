# operation MResetX(target : Qubit) : Result

## Summary
Measures a single qubit in the X basis,
and resets it to a fixed initial state
following the measurement.

## Description
Performs a single-qubit measurement in the X-basis,

following the measurement.

## Input
### target

## Output
